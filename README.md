### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 商品信息
5. 交流论坛
6. 公告资讯
7. 留言板
8. 后台管理
9. 购物车
10. 个人中心
11. 我的发布
12. 我的订单
13. 我的地址
14. 我的收藏


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 商品分类管理
5. 商品信息管理
6. 交流论坛
7. 留言板
8. 系统管理（轮播图管理、关于我们、公告资讯、系统简介）
9. 订单管理
